## Overview

This thesis project presents a road accident detection system focused on **Makati City**, leveraging **YOLOv12** trained on video footage enhanced using **Zero-3DCE** (Zero-Reference Deep Curve Estimation). The system detects two vehicle classes — **motorcycles** and **cars** — to identify and analyze road accidents in low-light and challenging visual conditions.

## Features

- Detects **cars** and **motorcycles** in real-time video
- Uses **Zero-3DCE** for low-light video enhancement before detection
- Focused on road accident scenarios in **Makati City**
- Powered by **YOLOv12** for fast and accurate object detection

## Model

- **Architecture:** YOLOv12
- **Classes:** Car, Motorcycle
- **Enhancement:** Zero-3DCE (Zero-Reference Deep Curve Estimation)
